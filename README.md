# 🗂️ ST Chat Organizer

Ультимативный менеджер чатов для SillyTavern. Наведите идеальный порядок в ваших переписках: группируйте чаты по персонажам, добавляйте теги, заметки, кастомные имена и раскрашивайте папки!

## ✨ Главные функции

### 🔍 Умная группировка и Поиск
* **Авто-папки:** Все чаты автоматически собираются в аккуратные складные папки по именам персонажей.
* **Глобальный поиск (Real-time):** Ищите нужный чат по оригинальному имени, кастомному названию, тексту заметок, тегам или имени бота. Папка с найденным чатом раскроется сама!
* **Живое автообновление:** Папки реагируют на события Таверны — они сами перестраиваются при создании, удалении или обновлении чата.

### 📁 Управление Папками
* **Счетчик чатов:** Точное количество переписок рядом с именем персонажа.
* **Цветовая кастомизация:** Задавайте уникальный цвет обводки и фона для папки каждого персонажа.
* **3 режима сортировки:** Внутри папки чаты можно выстроить по дате создания, по алфавиту или в **своем порядке** (Drag-and-Drop).

### 💬 Инструменты чата (Hover-панель)
При наведении на любой чат появляется удобная панель инструментов:
* 📌 **Закрепление (Пин):** Отправляет чат на самый верх папки.
* 🏷️ **Система тегов:** Добавляйте цветные плашки-теги для быстрой навигации.
* ✏️ **Кастомные имена:** Переименовывайте чаты визуально, не ломая оригинальные файлы.
* 📝 **Заметки к сюжету:** Оставляйте комментарии, превью которых будет видно прямо в списке.
* 🔢 **Счетчик сообщений:** Показывает точное количество реплик в переписке.
* 🖱️ **Drag-and-Drop:** Перетаскивайте чаты мышкой в нужном вам порядке.

### 📦 Массовые действия (Bulk Actions)
Выделяйте чаты чекбоксами (появляются слева от аватарки при наведении), чтобы вызвать верхнюю панель управления:
* Массовое присвоение тегов сразу всем выбранным чатам.
* Массовое закрепление/открепление (пины).
* Быстрый сброс выделения в один клик.

### ⚙️ Гибкие настройки и Бэкап
Меню расширения интегрировано прямо в интерфейс Таверны:
* **Дизайн:** Настраивайте глобальные цвета границ, базового фона, эффектов при наведении и прозрачность папок.
* **Экспорт/Импорт (Backup):** Скачивайте все ваши настройки, цвета, теги, заметки и порядок сортировки в `.json` файл, чтобы ничего не потерять при переустановке!

## 🛠️ Установка

1. Откройте SillyTavern.
2. Перейдите во вкладку **Extensions** (Расширения) -> **Install Extension**.
3. Вставьте ссылку на этот репозиторий:
   `(https://github.com/Gromova-Olga/ST-Chat-Organizer)`
4. Нажмите **Save & Connect**.
